1. Comment trong Python là gì?

Comment hay còn gọi là ghi chú, đây là những đoạn code mà trình biên dịch Python sẽ bỏ qua trong lúc chạy mã code.

Khi bạn phát triển dự án khổng lồ, hoặc bạn xây dựng một hàm nào đó thì cần lưu lại những lưu ý để sau này xem lại sẽ hiểu được những gì mà bạn đã code. Lúc này sử dụng cú pháp tạo comment sẽ giúp bạn giải quyết được vấn đề đó.
