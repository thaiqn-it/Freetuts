Ví dụ 1: Cộng hai số trong Java

Trong ví dụ này ta chỉ cộng hai số đã được gán sẵn giá trị, thực hiện phép tính cộng và hiển thị ra màn hình.

public class CongHaiSo{

   public static void main(String[] args) {

         

      int num1 = 5, num2 = 15, sum;

      sum = num1 + num2;

      System.out.println("Tổng của hai số là: "+sum);

   }

}

Ví dụ 2: Cộng hai số sử dụng Scanner

* Lưu ý: Để sử dụng được Scanner thì trước tiên ta phải import nó vào đã nhé, bằng cách sử dụng câu lệnh import java.util.Scanner.

Scanner cho phép chúng ta nắm bắt đầu vào của người dùng để chúng ta có thể nhận được các giá trị của cả hai số từ người dùng. Chương trình sau đó sẽ thực hiện tính tổng và hiển thị nó.

import java.util.Scanner;

public class CongHaiSo {

    public static void main(String[] args) {

        int num1, num2, sum;

        Scanner sc = new Scanner(System.in);

        System.out.println("Nhập vào số thứ nhất: ");

        num1 = sc.nextInt();

        System.out.println("Nhập vào số thứ hai: ");

        num2 = sc.nextInt();

        sc.close();

        sum = num1 + num2;

        System.out.println("Tổng của hai số vừa nhập là: "+sum);

    }

}

Kết quả:

Như vậy chúng ta vừa thực hiện xong hai chương trình tính tổng hai số bằng hai cách khác nhau đó là tính tổng của hai số đã được gán sẵn giá trị và hai số được nhập từ người dùng.
